Student Performance: Exploratory Data Audit
🔍 Project Overview
This repository demonstrates a foundational Data Integrity and Exploratory Data Analysis (EDA) audit. Using Python and the Pandas library, I performed a statistical validation of a student performance dataset to ensure high-quality data input for future AI modeling and reporting.
🛠️ Technical Skills Demonstrated
Data Structure Validation: Verifying schema and handling mixed datatypes.
Statistical Profiling: Identifying performance outliers and data anomalies.
Categorical Auditing: Analyzing demographic distributions to ensure data quality.
Schema Verification: Identifying null entries to prevent downstream processing errors.
📂 Files in this Repository
Student_Performace_Audit.ipynb: The complete technical report and Python logic.
study_performance.csv: The raw dataset used for the audit.
